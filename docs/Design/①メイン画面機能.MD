# ①メイン画面機能

## 機能一覧
|No|内容|
|---|---|
|1|タスク１〜６を表示|
|2|タスク１〜６の入力|
|3|タスク１〜６の優先順位を変更|
|4|タスク１〜６の状態を変更|
|5|表示する日を変更|
|6|昨日分の内容からタスクをコピー|
|7|昨日分の内容から未完了タスクをコピー|

## タスク１〜６を表示
- その日の日付を変数に保持
- 日付をキーにしてRealmファイルをロード
- ロードしたデータからタスク１〜６をTableView内のTextFieldに格納
- ロードしたデータからタスク１〜６状態をTableView内のチェックリストに反映

## タスク１〜６の入力
- TextFiledに入力する
- 桁幅を超えたら改行するようにしたい(できるか? TableViewCellサイズを可変にしないといけん)
